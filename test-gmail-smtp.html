1. Generate TempMail Address

https://temp-mail-i1gzgmfuu-randymay2178-1570s-projects.vercel.app

2. Test dengan Gmail (Tidak Akan Berhasil)

Jika Anda coba kirim email dari Gmail ke TempMail:

From: your-email@gmail.com
To: generated-email@tempmail.com
Subject: Test Email
Message: This is a test

Hasil: Email akan gagal terkirim karena domain tidak valid

3. Solusi untuk Bisa Terima Email Pribadi

Opsi A: Deploy ke Railway (Gratis)

1. Buka railway.app
2. Sign up dengan GitHub
3. Deploy TempMail
4. SMTP akan aktif di port 25

Opsi B: Deploy ke Render (Gratis)

1. Buka render.com
2. Sign up dengan GitHub
3. Deploy TempMail
4. SMTP akan aktif di port 25

Opsi C: Test Lokal

1. Jalankan: test-local-smtp.bat
2. SMTP aktif di localhost:2525
3. Test dengan email client

4. Test Lokal (Recommended)

Untuk test sekarang juga, jalankan:

.\test-local-smtp.bat

Kemudian:

  1. Generate email di dashboard
  2. Kirim email ke alamat tersebut
  3. Check inbox di dashboard

5. Konfigurasi Email Client

Untuk test lokal, gunakan setting ini:

SMTP Host: localhost
SMTP Port: 2525
SMTP Security: None
Authentication: None
